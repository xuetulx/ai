### 19. 审计日志与资源限制

#### 19.1 审计日志要求（增强版） `[DO]`

所有文件系统操作必须可审计、防篡改：

1. `[DO]` 审计日志必须包含以下信息：
   - 时间戳（精确到秒）
   - 操作类型（创建/读取/修改/删除/移动/重命名）
   - 目标文件/目录的完整路径
   - AI 模型版本和工具版本
   - 操作结果（成功/失败/部分成功）
2. `[DO]` 重要操作（删除、覆盖、批量操作）需记录操作前后的文件哈希或差异。
3. `[DO]` 审计日志应写入单独的日志文件（如 `.ai_audit.log`），与普通日志分离。
4. `[DO]` 审计日志不得被 AI 自行修改或删除，仅用于记录和追溯。

#### 19.2 资源限制 `[DO]`

防止 AI 意外耗尽系统资源（DoS 风险）：

1. `[DO]` 单任务生成的文件总数建议不超过 50 个（除非用户明确要求更多）。
2. `[DO]` 单个输出文件大小建议不超过 100MB（大数据文件除外，需用户确认）。
3. `[DO]` 批量操作时分批执行，每批不超过 5 个操作，避免瞬间大量 IO。
4. `[DO]` 长时间运行的任务应定期检查资源使用情况，发现异常及时停止并报告。
5. `[DO]` 临时文件总大小应控制在合理范围内，任务结束后立即清理。

#### 19.3 临时文件失败兜底 `[DO]`

1. `[DO]` 临时文件建议设置最大生命周期（如 24 小时），超时自动清理。
2. `[DO]` 任务异常中断时，下次启动应检查并清理遗留的临时文件。
3. `[DO]` 临时目录使用前检查磁盘空间，空间不足时提前警告。
4. `[DO]` 重要的中间结果建议保存为正式文件，而非仅存于临时目录。

### 19.4 依赖镜像源策略 `[DO]`

> 历史来源：AI 助手文件操作安全规范 v3.0 §5.3

#### 19.4.1 镜像源选择原则 `[DO]`

1. `[DO]` **默认使用国内镜像**加速下载，按以下优先级自动选择镜像源。
2. `[DO]` **镜像源选择优先级**：可用性检测 → 响应最快者优先，失败时自动切换。
3. `[ASK]` **修改项目镜像配置文件前询问用户**（`.npmrc`、`pip.conf`、`.cargo/config.toml`）。
4. `[DO]` **镜像超时时自动回退**至官方源，并通知用户。

#### 19.4.2 各语言默认镜像源

| 语言 | 镜像源 URL | 提供商 |
|------|-----------|--------|
| **Python (pip)** | `https://pypi.tuna.tsinghua.edu.cn/simple` | 清华大学（默认首选） |
| | `http://mirrors.aliyun.com/pypi/simple/` | 阿里云 |
| | `https://mirrors.cloud.tencent.com/pypi/simple` | 腾讯云 |
| | `https://repo.huaweicloud.com/repository/pypi/simple` | 华为云 |
| **Node.js (npm)** | `https://registry.npmmirror.com` | 淘宝/NPMMirror |
| | `https://mirrors.tencent.com/npm/` | 腾讯云 |
| | `https://repo.huaweicloud.com/repository/npm/` | 华为云 |
| **Go (go mod)** | `https://goproxy.cn` | 七牛云（Go 官方推荐） |
| **Rust (cargo)** | `https://mirrors.tuna.tsinghua.edu.cn/crates.io-index` | 清华大学 |
| **Java (Maven)** | `https://maven.aliyun.com/repository/public` | 阿里云 |

#### 19.4.3 安装命令示例

```bash
# Python pip 默认使用清华大学源（无需询问）
pip install <package> -i https://pypi.tuna.tsinghua.edu.cn/simple

# 清华大学源超时时自动回退到阿里云
pip install <package> -i http://mirrors.aliyun.com/pypi/simple/

# 再失败回退到腾讯云
pip install <package> -i https://mirrors.cloud.tencent.com/pypi/simple

# 全部镜像不可用时回退到官方源
pip install <package>
```

```bash
# Node.js npm 默认使用 NPMMirror（无需询问）
npm install <package> --registry https://registry.npmmirror.com
```
