# AI Skill 技能包说明

> 本目录存放 CodeBuddy 可用的 Skill 技能包，以 zip 格式存储，需安装后使用。

---

## 技能列表

### 1. Caveman（超压缩通信模式）
- **文件**: `caveman-main.zip`
- **功能**: 将 AI 输出切换为"原始人"风格，大幅压缩 token 用量（约节省 75%），同时保持技术准确性。
- **强度等级**: `lite` / `full`（默认）/ `ultra` / `wenyan-lite` / `wenyan-full` / `wenyan-ultra`
- **适用场景**: 需要高效沟通、减少 token 消耗时使用。

### 2. Find Skills（技能发现）
- **文件**: `find-skills-skill-1.0.0.zip`
- **功能**: 从多个来源搜索和发现 OpenClaw 生态中的可用技能。
- **适用场景**: 需要查找新技能、探索 CodeBuddy 能力扩展时使用。

### 3. FreeRide（免费 API Key）
- **文件**: `free-ride-1.0.11.zip`
- **功能**: 提供 OpenRouter API Key 获取途径（免费），可在 [openrouter.ai/keys](https://openrouter.ai/keys) 注册获取。
- **适用场景**: 需要通过 OpenRouter 调用多种 AI 模型时使用。

### 4. React Best Practices（React 最佳实践）
- **文件**: `react-best-practices.zip`
- **功能**: 来自 Vercel Engineering 的 React 和 Next.js 性能优化指南。涵盖组件编写、数据获取、打包优化等。
- **适用场景**: 编写、审查或重构 React/Next.js 代码时自动触发。

### 5. Self-Improvement（自我改进）
- **文件**: `self-improving-agent-3.0.23.zip`
- **功能**: 捕获错误、学习和修正。当命令失败、用户纠正错误、外部 API 失败等场景下触发，持续改进 AI 表现。
- **适用场景**: 需要 AI 从错误中学习并不断优化的长期项目。

### 6. Using Superpowers（技能系统指南）
- **文件**: `using-superpowers-0.1.0.zip`
- **功能**: 建立技能发现和加载的标准流程，确保 AI 在每次对话开始时正确识别和使用可用技能。
- **适用场景**: 所有对话的默认加载技能，保障技能系统正常运行。

---

## 安装方式

在 CodeBuddy 中通过 Skill 功能导入对应的 zip 文件即可使用。
